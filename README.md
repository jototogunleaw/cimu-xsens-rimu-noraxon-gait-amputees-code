# Naroxon_Xsens_Gait (Public Version)

## Overview
This repository contains the public-safe workflow structure for a gait analysis project using Xsens and Noraxon data.

The repository is shared to document the project pipeline, script organization, and overall processing workflow while some core algorithmic implementations are temporarily withheld.

## Included Files
This public version includes the following modules:

- `main_1_ProcessingXsensData.py`
- `main_2_ResultXsens.py`
- `main_3_ResultNaroxon.py`
- `selected.py`
- `divided.py`
- `addAllStep.py`
- `plotGraph.py`

## Withheld Files
The following modules are intentionally provided as placeholder versions only:

- `calculation.py`
- `detection.py`
- `normalization.py`

These files are part of the core signal-processing and gait-event workflow and are currently not included in full while copyright and ownership clearance is pending.

## Repository Purpose
This repository is intended to:

- document the overall analysis workflow
- present the script structure used in the project
- support academic transparency at the pipeline level
- protect proprietary or unpublished algorithmic implementations

## Notes
- This repository does **not** include raw datasets
- This repository does **not** include generated outputs or result files
- Some modules contain placeholder implementations and are therefore not executable in full

## Availability of Full Code
The complete implementation may be released after copyright clearance, publication, or approved request.
